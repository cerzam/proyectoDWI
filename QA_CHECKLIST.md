# Checklist de QA y Accesibilidad - Landing Page

## 📱 Responsividad (Diseño Adaptable)
- [ ] El componente `CTAForm` se adapta correctamente a pantallas móviles (320px).
- [ ] La navegación y las secciones no se desbordan horizontalmente.
- [ ] Los botones tienen un tamaño adecuado para tocarse en pantallas táctiles.

## ♿ Accesibilidad (a11y)
- [ ] Todos los inputs del `CTAForm` tienen una etiqueta `<label>` asociada.
- [ ] El contraste de colores entre el texto y el fondo es legible.
- [ ] Es posible navegar por la página y llegar al formulario usando únicamente la tecla `Tab` del teclado.

## ✅ Funcionalidad
- [ ] El formulario previene el envío si los campos obligatorios están vacíos.
- [ ] Los enlaces de navegación dirigen correctamente a sus secciones correspondientes.
